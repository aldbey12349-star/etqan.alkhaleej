# موقع شركة إتقان الخليج للخدمات المنزلية

هذا الموقع مصمم خصيصاً لشركة "إتقان الخليج" ليكون واجهة احترافية وفخمة، مع التركيز على تحويل الزوار إلى عملاء عبر الواتساب.

## كيفية تعديل محتوى الموقع بسهولة:
بدلاً من تعديل كل صفحة HTML، يمكنك تعديل ملف واحد فقط وهو:
`js/config.js`

### من خلال هذا الملف يمكنك تغيير:
1. **اسم الشركة:** غير قيمة `companyName`.
2. **اللوجو:** غير رابط `logoUrl`.
3. **أرقام التواصل:** غير `phone` و `whatsapp`.
4. **روابط التواصل الاجتماعي:** غير الروابط داخل قسم `social` (تيك توك، سناب شات، إنستجرام).
5. **الخدمات:** يمكنك إضافة أو تعديل أو حذف أي خدمة، وتغيير صورها ووصفها ومميزاتها.
6. **معرض الصور:** أضف صور أعمالك الجديدة في قسم `gallery`.

## المميزات التقنية:
- **تحويل البيانات للواتساب:** عند ملء أي نموذج طلب، يتم إرسال البيانات (الاسم، الجوال، المدينة، الخدمة، التفاصيل) مع **رابط موقع العميل الجغرافي تلقائياً** إلى رقم الواتساب الخاص بك.
- **تصميم متجاوب:** الموقع يعمل بكفاءة عالية على الجوال والتابلت والكمبيوتر.
- **القائمة الجانبية:** قائمة منسدلة ذكية تظهر في الجوال لتسهيل التنقل.
- **أزرار ثابتة:** زر واتساب وزر اتصال هاتفي يظهران دائماً في أسفل الصفحة.

## ملاحظات:
- الموقع جاهز للرفع مباشرة على أي استضافة.
- جميع الصور المستخدمة حالياً هي صور تجريبية (Placeholders)، يرجى استبدالها بروابط صورك الحقيقية من خلال ملف `js/config.js`.
- تم وضع علامة مائية (Watermark) باسم "إتقان الخليج" على صور الخدمات ومعرض الأعمال.
